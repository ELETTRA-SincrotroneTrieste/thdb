\doxysubsection{Detailed Description}
\begin{DoxyParagraph}{Description}
\doxyref{THdb\+Base}{p.}{classTHdbBase} is a class which you can directly use to connect to a My\+Sql database or subclass to personalize its behaviour and create a custom interface for your specific application.
\end{DoxyParagraph}
\begin{DoxyParagraph}{}
Create your \doxyref{THdb\+Base}{p.}{classTHdbBase} instance with a valid parent and then call set\+Type or set\+Connection\+Parameters to connect to a My\+Sql database. Then you can start registering your \doxyref{Db\+Query\+Interface}{p.}{classDbQueryInterface} implementation with register\+Db\+Query. You have to subclass \doxyref{Db\+Query\+Interface}{p.}{classDbQueryInterface} in order to place your query to the database via this framework. The method db\+Query\+Interface returns the \doxyref{Db\+Query\+Interface}{p.}{classDbQueryInterface} object with the name specified when register\+Db\+Query was called. Use unregister\+Db\+Query to remove a previously registered \doxyref{Db\+Query\+Interface}{p.}{classDbQueryInterface}. 
\end{DoxyParagraph}
