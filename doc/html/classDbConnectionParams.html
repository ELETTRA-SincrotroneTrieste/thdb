Detailed Description

Use this class to create a database connection or to switch an already active connection to another one with different parameters.

This class is used in conjunction with THdbBase::setType or THdbBase::setConnectionParameters.

Parameters
dbNamethe database name, or schema, such as "tdb", "hdb", "snap"...
dbUserthe user name to use to connect to the database
dbPasswordthe password to use to connect to the database
dbHostthe name of the host running the database, such as "hokuto", "hokuto.elettra.trieste.it"
dbPortthe tcp port on which the database server is listening to. The default value is 3306.

You can also construct a DbConnectionParams object with an empty constructor and then set the single parameters of the connection, which are simple public variables.
